Calculadoras de Premiação TCG

Selecione o formato do seu torneio para acessar a calculadora adequada.

1. Torneios Gerais

Distribuição focada na distribuição para o Top (Top 4, Top 8) utilizando boosters.

2. Pré-Release

Distribuição para Pré-Release garantindo boosters de participação para todos e separando o pool para o Top.

3. Pontuação (Boosters)

Distribuição proporcional de boosters calculada pela performance e pontos conquistados.

4. Pontuação (Crédito)

Distribuição monetária exata (em R$) baseada em pontos para gerar Crédito na Loja.
